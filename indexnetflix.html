Netflix Clone

O clone da Netflix é um projeto focado em apresentar o desenvolvimento
web como algo que faz parte do nosso dia dia,
pricipalmente durante nossas pesquisas nessa imensidão de
informação que é a internet, a fim de mostrar essa
interação produzimos um site clone da Netflix.
